Cyberhaven Triage — Análisis de Incidentes Asistido por IA

Clip | Jul 2025 – Actualidad

Construí un flujo automatizado de triage para incidentes DLP de alto riesgo. El sistema consulta incidentes en Cyberhaven, descarga archivos desde AWS S3, analiza el contexto con Gemini API y entrega los resultados en una interfaz Streamlit para revisión y retroalimentación del analista.

Stack técnico: Python, Streamlit, AWS EC2, AWS S3, Docker, Gemini API, Cyberhaven API

Impacto: Procesa 60+ incidentes diarios, aumentó la capacidad de atención del equipo 4x y redujo el tiempo de respuesta de días a horas.

Python AWS Streamlit Docker Gemini API Automatización
