# 🌐 Carlos Vertti - Personal Portfolio

Portafolio profesional bilingüe (ES/EN) con modo claro/oscuro, animaciones y proyectos reales.

## ✨ Características

- 🌓 Modo claro/oscuro con persistencia
- 🌍 Bilingüe (Español/Inglés)
- 🎨 Animaciones scroll (AOS)
- 📱 Diseño responsive
- ⚡ Performance optimizado
- 🚀 Deploy automático en GitHub Pages

## 🛠️ Stack

- HTML5 / CSS3
- JavaScript (Vanilla)
- Bootstrap 5.3
- AOS Library
- Inter Font
